4スタンス理論とは

人の動き方には
4つのタイプがある

TYPE A1

つま先・内側タイプ

横への体重移動が自然。ダイナミックな横系スイングで飛距離を生み出せる。

TYPE A2

つま先・外側タイプ

直立姿勢が基本。コンパクトで縦系の動きが得意。安定した再現性が持ち味。

TYPE B1

かかと・内側タイプ

TYPE B2

かかと・外側タイプ

広いベースで安定。力強いターンで飛距離を出すパワーヒッタータイプ。
