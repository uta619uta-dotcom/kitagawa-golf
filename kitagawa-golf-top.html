The 4-Stance Theory

4スタンス理論で
あなたのタイプを知る

A1・A2・B1・B2の4タイプ。それぞれの重心・軸・リリースに合わせた指導法で、無駄のないスイングを実現します。

A1
Type A1
A1タイプ

自然に立つと、重心が土踏まずのつま先側・内側へ自然と落ち込む。下半身の切り上げ動作でクラブが目標方向へ抜けるシャープなスイング。

A2
Type A2
A2タイプ

自然に立つと、重心が土踏まずのつま先側・外側へ自然と落ち込む。右半身から左半身へためなく切り上げるシンプルで無駄のないスイング。

B1
Type B1
B1タイプ

自然に立つと、重心が土踏まずのかかと側・内側へ自然と落ち込む。両足の同圧を保ちながら、右→左の背中引き込み動作で力強い捻転を生むスイング。

B2
Type B2
B2タイプ

自然に立つと、重心が土踏まずのかかと側・外側へ自然と落ち込む。両足の同圧を保ちつつ、みぞおちと肘の柔軟性でためを作る。北川自身のタイプ。
